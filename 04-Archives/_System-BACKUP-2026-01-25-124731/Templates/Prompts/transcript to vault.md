## 3. Implementation Guidelines

### A. File Naming Convention
- Use kebab-case for files
- Include version numbers
- Add context prefixes
Example: `tech-parallel-consciousness-debugging-v1-0-0.md`

### B. Cross-Reference Protocol
- Use full paths for links
- Include context notes
- Maintain bidirectional links
Example: `[[/Users/magenarayan/claudemcp/WhyChromosome-Vault/02-Areas/Tech-Parallels/consciousness-debugging.md]]`
