# Image Generation Prompts
`Version 1.0.0 | Visual Asset Creation Guide`

## Hero Visualization
```
Primary Prompt:
Create a mesmerizing 3D visualization of interlocking time wheels in motion. 
The scene features multiple translucent layers of geometric precision, with 
crystalline structures floating in a deep space environment. Include:
- Multiple spinning rings with sacred geometry patterns
- Floating particles that follow geometric paths
- Subtle energy streams connecting different elements
- Holographic UI elements hovering in space

Style Parameters:
- Color Palette: Deep space blacks to ethereal blues
- Lighting: Rim lighting with subtle volumetric effects
- Texture: Glossy and metallic with glass elements
- Mood: Technological mysticism

Technical Requirements:
- Resolution: 2400x1600px
- Style: Ultrarealistic 3D rendering
- Lighting: Volumetric, atmospheric
- Format: PNG with transparent background
```

## Animation Guidelines
```
For Dynamic Elements:
- Smooth, continuous rotation of time wheels
- Gentle floating motion for UI elements
- Pulsing energy effects for highlights
- Particle system movement paths
- Transition effects between states

Technical Specifications:
- Frame Rate: 60fps
- Format: WebM and MP4
- Duration: 10-second loops
- Resolution: 1920x1080px
```
