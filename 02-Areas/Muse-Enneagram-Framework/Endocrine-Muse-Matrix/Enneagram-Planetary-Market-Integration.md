### 1. New Moon to First Quarter (Waxing Crescent)
- **Phase**: Accumulation
- **Dominant Energy**: Type 5 (Investigation)

- **Market Characteristics**:
  - Research phase
  - Pattern identification
  - Position building
  - Low volatility
  - Institutional accumulation

### 2. First Quarter to Waxing Gibbous
- **Phase**: Mark Up
- **Dominant Energy**: Type 3 (Achievement)
- **Supporting Energy**: Type 7 (Opportunity)
- **Planetary Influence**: Jupiter
- **Market Characteristics**:
  - Trend establishment
  - Momentum building
  - FOMO activation

### 3. Full Moon
- **Phase**: Distribution
- **Dominant Energy**: Type 8 (Power)
- **Supporting Energy**: Type 2 (Sharing)
- **Planetary Influence**: Pluto/Moon
- **Market Characteristics**:
  - Peak prices
  - Maximum volatility
  - Emotional trading
  - Volume climax
  - Smart money distribution

### 4. Waning Gibbous to Last Quarter
- **Phase**: Mark Down
- **Dominant Energy**: Type 6 (Security)
- **Supporting Energy**: Type 4 (Reflection)
- **Planetary Influence**: Mars/Neptune
- **Market Characteristics**:
  - Profit taking
  - Risk assessment
  - Defensive positioning
  - Volume decline
  - Price correction

### 5. Last Quarter to New Moon
- **Phase**: Consolidation
- **Dominant Energy**: Type 9 (Integration)
- **Supporting Energy**: Type 5 (Analysis)
- **Planetary Influence**: Venus/Saturn
- **Market Characteristics**:
  - Base building
  - Pattern completion
  - Low volatility
  - Volume stabilization
  - Preparation for new cycle
