# Channel Recipes

## Substack (longform)
- Length: 1200-2200 words
- Structure: Hook → Runtime problem → Pattern method → Enneagram lens → Embodied protocol → Non-pitch close
- Must include one practical protocol block

## X Thread
- Length: 8-13 tweets
- Structure: Pattern interrupt → problem → layered reframe → protocol → question
- Must include one clear embodied micro-action

## LinkedIn
- Length: 220-450 words
- Structure: sharp opening + 3-point framework + practical takeaway
- Tone: professional, grounded, less esoteric density

## Instagram Caption/Carousel Script
- 5-slide or 7-slide logic
- Each slide: one core claim + one line embodiment cue
- Avoid jargon without contextual translation
