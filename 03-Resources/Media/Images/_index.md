## Usage Guidelines

### Adding New Images
1. Determine appropriate category
2. Follow naming conventions
3. Update relevant index files
4. Document usage in source files

### Linking Images
- Use relative paths: `../../../03-Resources/Media/Images/Category/filename.ext`
- Include descriptive alt text
- Document dependencies

### File Management
- Optimize file sizes when necessary
- Use appropriate formats (PNG for diagrams, JPEG for photos)
- Create metadata files for generated content
- Archive rather than delete deprecated images
