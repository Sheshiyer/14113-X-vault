## Usage Guidelines

### File Naming Conventions
- **Generated Content**: Preserve original names with metadata files
- **Screenshots**: `screenshot_YYYY-MM-DD_description.ext`
- **Diagrams**: `diagram_system-name_description.ext`
- **Articles**: `article_title_image-purpose.ext`
- **Brand**: `brand_asset-type_description.ext`

### Linking Protocols
- Use relative paths from document location
- Maintain descriptive alt text for accessibility
- Document media dependencies in source files
- Update links when moving or renaming files

### Organization Standards
- Place files in appropriate category directories
- Use descriptive filenames when possible
- Create metadata files for generated content
- Archive deprecated content rather than deleting
