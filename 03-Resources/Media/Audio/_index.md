## Usage Guidelines

### Adding New Audio
1. Determine appropriate category
2. Follow naming conventions
3. Update this index file
4. Document usage in source files
5. Consider file size and quality

### Linking Audio
- Use relative paths from document location
- Include descriptive captions
- Document audio dependencies
- Consider accessibility requirements (transcripts)

### File Management
- Monitor file sizes and quality
- Use appropriate formats (MP3 for compatibility, WAV for quality)
- Create metadata files for generated content
- Archive rather than delete deprecated audio
