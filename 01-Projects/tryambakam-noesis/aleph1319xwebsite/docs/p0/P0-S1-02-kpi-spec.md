# P0-S1-02 KPI Specification

Date: 2026-02-16
Owner: Product Strategist
Status: Drafted

## North Star
Increase practitioner progression across ecosystem layers, not pageviews.

## Primary KPIs
1. Orientation Completion Rate
Definition: Sessions that visit at least 3 ecosystem layers.
Target (beta): >= 45%

2. Cross-Layer Transition Depth
Definition: Mean count of distinct layer transitions per session.
Target (beta): >= 2.2

3. Initiation Start Rate
Definition: Sessions that begin `.init` pathway from any entry surface.
Target (beta): >= 12%

4. Qualified Interest Conversion
Definition: Sessions resulting in at least one qualified action (preview request, DM trigger, mentorship application, Kopina interest signal).
Target (beta): >= 8%

5. Ecosystem Return Rate (7-day)
Definition: Users returning within 7 days after first meaningful session.
Target (beta): >= 22%

## Secondary KPIs
- Handoff click-through to Selemene/TUI/1319 references
- Layer-specific dwell quality score
- Mentorship application completion rate
- Kopina technical page completion and intent submission

## Guardrail KPIs
- Fallback-mode usage (WebGL unsupported) <= 8%
- Accessibility critical issues = 0
- P95 API latency <= 300ms
