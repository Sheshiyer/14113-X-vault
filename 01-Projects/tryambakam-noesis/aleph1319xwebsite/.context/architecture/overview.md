# Architecture Overview

- Product mode: ecosystem orientation portal across all Noesis layers.
- Explicit non-goal: no engine calculation frontend in this codebase.
- Rendering model: React Three Fiber canvas with chunked partitions and culling.
- Content model: graph-driven layers, nodes, and integration links.
- Platform model: Next.js frontend, Worker API, Supabase data/realtime.
- Brand model: Noesis visual system and technical Kopina terminology.
