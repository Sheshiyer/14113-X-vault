## Overview

The Shadow-Gift-Siddhi triad represents **three frequencies of consciousness** through which each of the 64 Gene Keys expresses. This is the core transformative framework of the Gene Keys system.

**Key Principle:** All three frequencies are valid and necessary. Shadow is not "bad" - it's the pressure that catalyzes transformation.

**Key Insight:**
- Siddhi is not achievable through effort
- It's a state of grace
- Often spontaneous, temporary
- Living AS the archetypal perfection

### Key Structure Elements

1. **Number** (1-64)
   - Corresponds to I-Ching hexagram
   - Maps to Human Design gate
   - Unique archetypal identifier

2. **Name** (The Creative, The Orientation, etc.)
   - Archetypal essence
   - Often from I-Ching translation
   - Poetic/symbolic

3. **Shadow-Gift-Siddhi** (Strings)
   - Single-word frequencies
   - Transformation pathway
   - Core teaching distilled

4. **Codon** (Three-letter DNA code)
   - Links to genetic codons
   - 64 codons ‚Üí 64 Gene Keys
   - Maps to amino acids

5. **Programming Partner** (Integer 1-64)
   - Complementary Gene Key
   - Creates codon pair
   - Dynamic balance relationship

6. **Descriptions** (Text)
   - Detailed frequency explanations
   - Practical manifestations
   - Guidance for working with each

7. **Life Theme** (String)
   - Core life lesson
   - Overarching purpose
   - Integration statement

## Practical Implementation

### Loading Gene Key Data

```python
def _get_gene_key_by_number(self, number: int) -> GeneKey:
    """Get Gene Key by its number."""
    # Ensure number is within valid range
    if number < 1 or number > 64:
        number = ((number - 1) % 64) + 1
    
    # If Gene Key doesn't exist in data, use fallback
    if str(number) not in self.gene_keys_data.gene_keys:
        number = 1
    
    key_data = self.gene_keys_data.gene_keys[str(number)]
    return GeneKey(**key_data)
```

**Safety Features:**
- Modulo operation wraps out-of-range numbers
- Fallback to Gene Key 1 if data missing
- Robust against data errors
