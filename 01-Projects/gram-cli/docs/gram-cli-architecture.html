glam-cli Architecture Overview
Instagram media download CLI with cookie auth, reliability-critical safeguards, and automation-oriented behavior.

1) Business Context

Objective

Enable authenticated and public Instagram downloads from a scriptable CLI with predictable outcomes for automation workflows.

Primary Users

Developers, archivists, and operators running recurring content backup or ad-hoc fetch jobs.

Value Delivered

Single tool for login, profile/post/stories/highlights downloads, and machine-readable output modes.

Reliability Metrics

  • Errors return non-zero exit codes.
  • Resume mode skips already-downloaded post artifacts.
  • Test suite validates critical command semantics.

6) Deployment & Workflows

Release Artifacts

Automation Contract

  • Resume mode avoids duplicate post downloads

Operational Prerequisites
